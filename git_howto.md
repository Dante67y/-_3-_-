# Инструкция по работе с GIT

## Основные команды GIT

1. git init
```sh
инициализация локального репозитория
```
2. git status
```sh
получить информацию от git о его текущем состоянии
```
3. git add
```sh
добавить файл или файлы к следующему коммиту
```
4. git commit -m (message)
```sh
создание коммита
```
