# Подсказка по командной строке

## Как открыть терминал?

1. Win + R
2. Прописываем cmd.exe
3. Нажимаем "Ok"
4. Внутри прописываем "cd" и указываем путь к папке

## Как сменить директорию
### Windows
```sh
cd C:\folder_name
```

### MacOc/Linux
```sh
cd /folder_name
```

## Листинг директории
### Windows
```sh
dir
```

### MacOs/Linux
```sh
ls
```

## Как выйти из VIM(commit)
### Без сохранения
```sh
Esc
:q!
```

### С сохранением
```sh
Esc
:wq
```

## Как удалить файл
### Windows
```sh
del file_name
```

### MacOs/Linux
```sh
rm file_name
```
